# Security Policy

## Scope

CyberForge Lab is an educational platform for isolated, authorised cybersecurity labs only.

## Reporting a Security Issue

If you find a security issue in this repository:

1. Do not publish sensitive details publicly.
2. Open a private report through repository security reporting if enabled.
3. Include clear reproduction steps and affected files.

## Out of Scope

- Requests for offensive automation against public systems.
- Real-world credential theft or persistence techniques.
- Any unauthorised target testing guidance.

## Safe Handling Rules

- Never commit secrets.
- Never add real credentials.
- Keep `.test` or `localhost` domains only in scenarios.
- Keep `internet_access: false` for initial scenarios unless reviewed.
