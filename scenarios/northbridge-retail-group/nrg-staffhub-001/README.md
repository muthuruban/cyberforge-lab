# nrg-staffhub-001

## Scenario Title

Suspicious Activity in the Northbridge StaffHub Portal

## Story Context

Northbridge Retail Group is a fictional retail and e-commerce organisation. The security team has observed unusual login behavior and administrative access patterns in the StaffHub Portal. The learner must investigate app behavior and logs, then produce a professional finding.

## Learning Focus

- authentication weakness investigation
- application log review
- secure coding remediation planning
- SOC-style reporting

## Local Access

- Application URL: `http://127.0.0.1:8088`
- Service name: `staffhub`
- Container name: `cyberforge-nrg-staffhub`

## Safety Notice

This scenario is lab-only and educational. Do not repurpose for unauthorised testing.
